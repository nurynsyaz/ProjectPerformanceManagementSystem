# 📊 Project Performance Management System (PPMS)
This is my Final Year Project for the Bachelor of Computer Science (Software Engineering) with Honours at Universiti Malaysia Terengganu, developed by Nurin Syazwani Binti Faizal Azmir.

Project Purpose:
The Project Performance Management System (PPMS) is a comprehensive web-based application designed to efficiently manage, assign, and monitor the performance of projects within an organization. It supports user role-based task tracking, progress reporting, team collaboration, and comment management—making project oversight more transparent and productive.
